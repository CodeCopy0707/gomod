### 🚀 **Usage Instructions**

Once deployed, users can:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/CodeCopy0707/gomod.git
   cd gomod
   ```

2. **Quick Setup**:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

3. **Verify Installation**:
   ```bash
   chmod +x verify_installation.sh
   ./verify_installation.sh
   ```

4. **Run Demo**:
   ```bash
   go run demo.go
   ```

5. **Start Agent**:
   ```bash
   ./ai-agent
   ```
